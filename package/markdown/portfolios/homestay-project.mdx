## Project Overview

Ami Homestay is a modern web platform designed to help travelers discover and book cozy, family-friendly accommodations. The site offers a warm user experience, inviting visitors to explore rooms, check gallery images, and book directly through the platform.

## Features

- Holiday accommodation details  
- Room categories and gallery pages  
- Interactive navigation with dropdowns  
- Booking integration with clear call-to-actions  
- Contact information and address display  
- Mobile-responsive and visually inviting layout

## Technology Stack

- React / Next.js  
- Tailwind CSS  
- Responsive Design  
- Static Site Generation (SSG)  
- Email contact and inquiry-ready sections
